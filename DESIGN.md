# AlphaBrain Design System
**Version:** 1.0
**Last Updated:** 2025-04-13

---

## Brand Personality

AlphaBrain is a warm, focused learning companion. Not generic AI, not corporate SaaS.

**Keywords:** Focus, Clarity, Progress, Warmth

---

## Typography

### Font Families

Headings: Space Grotesk (Google Fonts)
- Purpose: Display, distinctive, modern
- Weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)

Body: DM Sans (Google Fonts)
- Purpose: Readable, neutral, clean
- Weights: 400 (regular), 500 (medium), 600 (semibold)

Code: JetBrains Mono
- Purpose: Technical content, code snippets

### Type Scale

- H1: 48px / 1.1 - Page titles
- H2: 36px / 1.2 - Section titles
- H3: 24px / 1.3 - Subsection titles
- H4: 18px / 1.4 - Card titles
- Body: 16px / 1.5 - Body text
- Small: 14px / 1.5 - Captions

## Color Palette

### Neutrals (Warm Base)

- gray-50: #FAFAF9 (backgrounds)
- gray-100: #F5F5F4 (cards)
- gray-200: #E7E5E4 (borders)
- gray-300: #D6D3D1 (subtle dividers)
- gray-400: #A8A29E (disabled)
- gray-500: #78716C (secondary text)
- gray-700: #44403C (primary text)
- gray-900: #1C1917 (headings)

### Accent (Coral)

- coral-500: #F97316 (primary action)
- coral-600: #EA580C (hover)
- coral-400: #FB923C (subtle accent)

### Semantic Colors

- Success: #059669 (green-600)
- Warning: #D97706 (amber-600)
- Error: #DC2626 (red-600)
- Info: #0284C7 (sky-600)

### Event Types (New Palette)

- Deep Work: #F97316 (coral)
- Review: #059669 (green)
- Planning: #0284C7 (sky)
- Creative: #D97706 (amber)
- Research: #7C3AED (violet - sparing use)

## Spacing

Base unit: 4px

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px

---

## Border Radius

- sm: 4px (buttons, small inputs)
- md: 8px (cards, larger inputs)
- lg: 12px (modals, panels)
- full: 9999px (pills, badges)

---

## Components

### Buttons

Primary: coral-500 background, white text
Secondary: gray-100 background, gray-700 text
Ghost: transparent, gray-700 text

### Cards

Background: gray-50
Border: gray-200
Radius: md (8px)
Shadow: sm

### Inputs

Border: gray-300
Focus: coral-500 ring-2
Radius: sm (4px)

---

## Accessibility

- Focus rings: 2px coral-500
- Contrast: WCAG AA (4.5:1) minimum
- Touch targets: 44px minimum
